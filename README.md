# yenepay.sdk.react-native

YenePay sdk for react native

## Installation

```sh
npm install yenepay.sdk.react-native
```

## Usage

```js
import YenepaySdkReactNative from "yenepay.sdk.react-native";

// ...

const result = await YenepaySdkReactNative.multiply(3, 7);
```
